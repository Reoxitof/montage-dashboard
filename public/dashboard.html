🔑 Mon Token Twitch
Pourquoi enregistrer ton token ?
Sans token, les sanctions (timeout/ban) apparaissent sous le nom du bot.
Avec ton token, elles apparaissent sous ton propre nom Twitch.
📋 COMMENT RÉCUPÉRER TON TOKEN
  1. Va sur twitchtokengenerator.com
  2. Clique sur Bot Chat Token
  3. Connecte-toi avec ton compte Twitch
  4. Autorise l'application
  5. Copie le champ Access Token et ajoute oauth: devant
  6. Colle-le ci-dessous et clique Enregistrer
⚠️ IMPORTANT — SÉCURITÉ
  • Ce token donne accès à modérer le chat en ton nom uniquement
  • Il ne donne pas accès à ton mot de passe, email ou paiements
  • Tu peux le révoquer à tout moment sur twitch.tv/settings/connections
  • Il est stocké de façon sécurisée et visible uniquement par l'admin
